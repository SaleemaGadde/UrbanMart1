# UrbanMart1 – E-Commerce Web Application

Overview

UrbanMart1 is a full-stack e-commerce web application developed using Spring Boot. It provides users with a simple and responsive online shopping experience, allowing them to browse products, add items to a cart, place orders, and manage payments.

## Features
- Product Listing
- Add to Cart
- Update Cart Quantity
- Remove Items from Cart
- Shopping Cart Total Calculation
- Checkout Process
- Payment Module
- Responsive User Interface
- Database Integration with MySQL

## Installation

1. Clone the repository.
2. Open the project in Eclipse or IntelliJ IDEA.
3. Configure MySQL and update the database details in "application.properties".
4. Run the Spring Boot application.
5. Open your browser and visit:  http://localhost:8082/
